Module Hierarchy

  • gui: GUI manager for Blender.

Class Hierarchy

  • __builtin__.object: The most base type
    • gui.Container: Geometry manager (base, abstract class).
      • gui.Grid: Grid geometry manager.
      • gui.Panels: Container manager (Container subclass)
      • gui.Rows: Row geometry manager (Container subclass).
      • gui.SingleWidget: Single widget geometry manager (Container subclass).
      • gui.Tabs: Container manager (Container subclass)
    • gui.Interface: GUI manager.
    • __builtin__.type: type(object) -> the object's type type(name, bases, dict) -> a new type
    • gui.Widget: Widget (base, abstract class)
      • gui.Button: Push Button (Widget subclass).
      • gui.LargeText: Display block of text (L{Widget} subclass).
      • gui.MultipleSelect: Multiple select (abstract class, Widget subclass).
        • gui.Menu: Menu (MultipleSelect subclass)
        • gui.MultipleToggle: Multiple toggle (MultipleSelect subclass).
        • gui.RadioButtons: Radio buttons (MultipleSelect subclass).
      • gui.Number: Number input base, abstract class (Widget subclass).
        • gui.FloatNumber: Floating-point number input (Number subclass).
        • gui.FloatSlider: Floating-point number input (Number subclass)
        • gui.IntNumber: Integer number input (Number subclass).
        • gui.IntSlider: Integer number input (Number subclass).
      • gui.String: String input (Widget subclass).
      • gui.Text: Static text (Widget subclass).
      • gui.Toggle: Toggle Button (Widget sublcass).
