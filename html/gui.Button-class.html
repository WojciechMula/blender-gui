Type Button

object --+    
         |    
    Widget --+
             |
            Button

Push Button (Widget subclass). Callback is called when user press the button.

Following attributes are defined:
  • name (read/write) - title of button
  • tooltip (read/write) - tooltip
Example: 
       # file: ex_Button.py
       import Blender
       import gui

       # define callbacks
       def hide_me():
               button1.show(False)
               Blender.Draw.Redraw()

       clicks = 0
       def count_click():
               global clicks
               clicks += 1
               button2.name = "Clicked %d time(s)" % clicks

       # make interface
       interface = gui.Interface()

       button1 = gui.Button(interface, 'Hide me!', callback=hide_me)
       button2 = gui.Button(interface, 'This button has no clicked yet.', callback=count_click)
       row = gui.Rows(interface, 10,10, 250, 20)
       row.addrow( [button1, (button2,0.7)] )

       interface.register_container(row)
       interface.run()
       # eof
