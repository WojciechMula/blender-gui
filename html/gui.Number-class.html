Known Subclasses:
FloatNumber, FloatSlider, IntNumber, IntSlider

Number input base, abstract class (Widget subclass).

Type of number is integer or floating-point, depending on subclass. Callback is called on value change.

Following attributes are defined:
  • name (read/write) - title of input
  • tooltip (read/write)
  • value (read/write) - current value
  • min (read only) - min allowed value
  • max (read only) - max allowed value
